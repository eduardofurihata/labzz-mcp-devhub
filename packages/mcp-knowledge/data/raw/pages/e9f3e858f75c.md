# Listar matrículas do aluno

Retorna a lista de matrículas de um aluno específico em uma escola.

**Required scopes:**alpaclass\_student\_read

GET

https://api.eduzz.com/alpaclass/v1/producer/teams/:teamId/students/:studentId/enrollments

### Path params

**teamId**

Id da escola

**studentId**

Id do aluno

### Query params

**page**

number

Página atual

**itemsPerPage**

number

Número máximo de itens por página

### Response params (200)

**pages**

number

Número de páginas

**page**

number

Página atual

**itemsPerPage**

number

Número máximo de itens por página

**totalItems**

number

Número total de items

**items**

array of object

Lista de matrículas do aluno

**items\[n\].id**

string

Id da matrícula

**items\[n\].createdAt**

string

Data de criação da matrícula

**items\[n\].certificateUrl**Nullable

string

URL para download do certificado. Pode ser nulo se não estiver disponível.

**items\[n\].progress**

number

Progresso do aluno no curso (em porcentagem)

**items\[n\].course**

object

Detalhes do curso

**items\[n\].course.id**

string

Id do curso

**items\[n\].course.name**

string

Nome do curso

### Status codes

Status

Descrição

200

Success
