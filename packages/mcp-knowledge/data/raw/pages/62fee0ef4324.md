# Tentativa de cobrança de fatura de assinatura via Saldo Eduzz

Notifica quando uma tentativa de cobrança de fatura de assinatura via Saldo Eduzz é realizada.

**Evento:**

myeduzz.contract\_eduzz\_balance\_attempted

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento

**data**

object

Dados do evento

**data.producer**

object

Dados do produtor

**data.producer.id**

string

ID do produtor

**data.producer.name**

string

Nome do produtor

**data.producer.email**

string

E-mail do produtor

**data.invoice**

object

Dados da fatura

**data.invoice.id**

string

ID da fatura

**data.invoice.payment**

object

Dados do pagamento

**data.invoice.payment.method**

enum

Método de pagamento

eduzzBalance

Saldo Eduzz

**data.invoice.failReason**Nullable

enum

Código do motivo da falha (se houver)

SA\_EMLPS

Operadora recusou a compra

SA\_VLAVL

Saldo insuficiente

**data.invoice.failReasonMessage**Nullable

string

Descrição do motivo da falha (se houver)

**data.invoice.isNegotiation**

boolean

Indica se é uma negociação

**data.invoice.status**

enum

Status da fatura

open

Fatura aberta

processing

Pagamento em processamento

paid

Fatura paga

canceled

Fatura cancelada

waitingDocuments

Aguardando envio de documentos

waitingRefund

Aguardando reembolso

refunded

Fatura reembolsada

analysing

Pagamento em análise

duplicated

Fatura duplicada

expired

Fatura expirada

recovering

Em recuperação

internal

Uso interno

trial

Período de teste (trial)

deleted

Fatura deletada

waitingPayment

Aguardando pagamento

refused

Pagamento recusado

overdue

Fatura vencida

scheduled

Pagamento agendado

negotiated

Fatura negociada

partialRefund

Reembolso parcial

**data.invoice.dueDate**

string

Data de vencimento da fatura

**data.invoice.attemptDate**

string

Data da tentativa de cobrança

**data.contract**

object

Dados do contrato

**data.contract.id**

string

ID do contrato

**data.contract.payment**

object

Dados do pagamento do contrato

**data.contract.payment.method**

enum

Método de pagamento do contrato

bankslip

Boleto Bancário

creditCard

Cartão de Crédito

pix

Pix

eduzzBalance

Saldo Eduzz

**data.contract.status**

enum

Status do contrato

upToDate

Em dia

awaitingPayment

Aguardando pagamento

late

Atrasado

canceled

Cancelado

defaulter

Inadimplente

suspended

Suspenso

trial

Trial

finished

Finalizado

free

Gratuito

**data.contract.createdAt**

string

Data de criação do contrato

**data.contract.updatedAt**Nullable

string

Data de atualização do contrato

**data.customer**

object

Dados do cliente

**data.customer.id**

string

ID do cliente

**data.customer.name**

string

Nome do cliente

**data.customer.email**

string

E-mail do cliente

**data.sentDate**

string

Data do envio do evento
