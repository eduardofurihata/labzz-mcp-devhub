# Em recuperação

Notifica quando o status da fatura muda para em recuperação

**Evento:**

myeduzz.invoice\_recovering

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento

**data**

object

Dados da fatura, produtor, comprador

**data.id**

string

**data.status**

string

**data.buyer**

object

Dados do comprador

**data.buyer.id**

string

Id do comprador

**data.buyer.name**

string

Nome do comprador

**data.buyer.document**

string

Documento do comprador

**data.buyer.email**

string

Email do comprador

**data.buyer.phone**Nullable

string

string

Telefone do comprador

**data.buyer.cellphone**Nullable

string

Celular do comprador

object

Endereço do comprador

**data.buyer.address.street**Nullable

string

Nome da rua

**data.buyer.address.number**Nullable

string

Número do logradouro

**data.buyer.address.neighborhood**Nullable

string

Nome do bairro

**data.buyer.address.complement**Nullable

string

Complemento do endereço

**data.buyer.address.city**Nullable

Nome da cidade

**data.buyer.address.state**Nullable

string

object

Preço da fatura

string

Moeda da fatura

number

Valor total da fatura

**data.orderBump**

object

Dados relacionados à OrderBump

boolean

**data.orderBump.isMainSale**

boolean

object

**data.items.coupon.discount.currency**

string

Moeda do desconto aplicado

**data.items.coupon.discount.value**

number

Valor do desconto aplicado

**data.items.partnerId**Nullable

string

Id do parceiro

**data.items.billingType**

Recurrence

Single

Other

Unknown

number

**data.checkoutUrl**

string

URL do Checkout

**data.bankslipUrl**Nullable

string

Url para acesso ao boleto caso seja um pagamento via boleto

**data.paymentMethod**

enum

Método de pagamento

bankslip

Bankslip

pix

Pix

Credit Card

combinedPayment

Combined Payment

installmentBankslip

Installment Bankslip

unknown

Unknown

**data.postback**Nullable

string

Url de postback configurada

**data.saleRecoveryUrl**Nullable

string

Url de recuperação da venda

**data.startRecoveringAt**

string

Data de início da recuperação

**data.bankSlipInstallment**Nullable

object

Informações de parcelas do boleto, caso o pagamento seja parcelado via boleto

**data.bankSlipInstallment.installmentNumber**

number

Número da parcela

**data.bankSlipInstallment.totalInstallments**

number

Total de parcelas

**data.contract**Nullable

object

Contrato da fatura

**data.contract.id**

string

Id do contrato

**data.contract.isUnlimitedInstallments**

boolean

Indica se o contrato é PSL (parcelamento sem limite)

**data.payment**

object

Dados de pagamento da fatura

**data.payment.method**

string

Método de pagamento utilizado

**data.payment.details**

string

Detalhes adicionais do pagamento

**sentDate**

string

Data do envio do evento
