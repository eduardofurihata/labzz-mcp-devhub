# Listar detalhes da entrega

Listar os detalhes da entrega

**Required scopes:**alpaclass\_delivery\_read

GET

https://api.eduzz.com/alpaclass/v1/producer/teams/:teamId/deliveries/:deliveryId

### Path params

**teamId**

Id da escola

**deliveryId**

ID da entrega

### Response params (200)

**id**

string

Id da entrega

**name**

string

Nome da entrega

**platform**

enum

Nome da plataforma

eduzz

Eduzz

dmg

Digital Manager Guru

hotmart

Hotmart

kiwifi

Kiwify

tmb

TMB

api

API

**platformProductId**

string

ID do produto na plataforma

**platformOfferId**Nullable

string

Código da oferta

**isActive**

boolean

Identifica se a entrega esta ativa

**isSubscription**

boolean

Identifica se a entrega é assinatura

**autoRemoveAccess**

boolean

Remove acesso automaticamente

**tags**

array of string

Tags cadastradas na entrega

**endsAfter**

number

Tempo de acesso em dias

### Status codes

Status

Descrição

200

Success
