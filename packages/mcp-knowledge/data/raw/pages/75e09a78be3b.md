# Vincular Participantes

Atribui um marcador a um ou mais participantes

**Required scopes:**blinket\_read

**Rate limit:**30 requests/minute

POST

https://api.eduzz.com/blinket/v1/attendance-tags/:id/attendance-list

### Path params

**id**

Id do marcador

### Body params

**attendanceIds**

array of string

Ids dos participantes que serão vinculados ao marcador

### Status codes

Status

Descrição

204

Success

403

Este usuário não tem permissão

422

Evento encerrado
