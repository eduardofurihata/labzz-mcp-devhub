# Matrícula Criada

Notifica quando um Aluno é matriculado em um Curso

**Evento:**

nutror.enrollment\_created

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento (ex: certificate\_viewed)

**sentDate**

string

Data de envio
