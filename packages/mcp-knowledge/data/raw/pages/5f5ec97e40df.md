# Listar cursos da entrega

Listar os cursos da entrega

**Required scopes:**alpaclass\_delivery\_read

GET

https://api.eduzz.com/alpaclass/v1/producer/teams/:teamId/deliveries/:deliveryId/courses

### Path params

**teamId**

Id da escola

**deliveryId**

ID da entrega

### Query params

**page**

number

Página atual

**itemsPerPage**

number

Número máximo de itens por página

**sort**

string

Ordenação dos itens podendo ter várias ordenações, seguindo o seguinte padrão: campo:ordenação Ex.: name:asc,platform:desc

### Response params (200)

**pages**

number

Número de páginas

**page**

number

Página atual

**itemsPerPage**

number

Número máximo de itens por página

**totalItems**

number

Número total de items

**items**

array of object

Cursos da Entrega

**items\[n\].id**

string

Id do curso

**items\[n\].name**

string

Nome do curso

### Status codes

Status

Descrição

200

Success
