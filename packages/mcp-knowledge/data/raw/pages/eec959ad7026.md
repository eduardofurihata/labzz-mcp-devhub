# Remover Marcador

Remove um marcador e o desvincula de todos os participantes que o possuem

**Required scopes:**blinket\_write

**Rate limit:**30 requests/minute

DELETE

https://api.eduzz.com/blinket/v1/attendance-tags/:id

### Path params

**id**

Id do marcador

### Status codes

Status

Descrição

204

Success

403

Este usuário não tem permissão

404

Marcador ou participante não encontrado

[![Swagger Logo](/_next/image?url=%2Flogos%2Fswagger.webp&w=256&q=75 "Swagger API Reference")](/swagger#/Blinket/deleteblinketv1attendancetagsid)

### Exemplos

ShellJavaScript

```shell
1curl --request DELETE \
2	--url 'https://api.eduzz.com/blinket/v1/attendance-tags/:id' \
3	--header 'Accept: application/json' 
```

**Response**204 - Marcador removido e participantes deletados
