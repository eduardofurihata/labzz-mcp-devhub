# Listar Eventos

Uma lista completa dos seus eventos

**Required scopes:**blinket\_read

**Rate limit:**30 requests/minute

GET

https://api.eduzz.com/blinket/v1/events

Lista seus eventos na blinket. Você pode filtrar por data, status, tipo de evento e etc.

### Query params

**page**

number

Página atual

**itemsPerPage**

number

Número de itens por página

**order**

enum

Ordenação dos resultados (ascendente ou descendente)

asc

Ascendente

desc

Descendente

**type**

enum

Tipo do evento

online

On-line

presential

Presencial

live

Ao vivo

**search**

string

Busca por Termo

**startDate**

datetime

Data de início do evento

**endDate**

datetime

Data do final do evento

### Response params (200)

**pages**

number

Número de páginas

**page**

number

Página atual

**itemsPerPage**

number

Número máximo de itens por página

**totalItems**

number

Número total de items

**items**

array of object

**items\[n\].userId**

**items\[n\].description**

**items\[n\].transmissionLinks**

object
