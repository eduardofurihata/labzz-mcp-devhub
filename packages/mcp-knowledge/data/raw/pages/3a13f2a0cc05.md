# Ingresso editado

Notifica quando é realizada uma alteração em um ingresso ou lote

**Evento:**

blinket.ticket\_edited

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento (ticket\_edited)

**sentDate**

string

Data do envio do evento

**data**

object

Dados do evento, produtor e ticket

**data.producer**

object

Dados do produtor

**data.producer.id**

number

Id do produtor do evento

**data.producer.name**

string

Nome do produtor do evento

**data.producer.email**

string

Email do produtor do evento

**data.event**

object

Dados do evento

**data.event.id**

string

Id do evento

**data.event.name**

string

Nome do evento

**data.event.type**

string

Tipo do evento

**data.tickets**

array of object

Dados dos ingressos

**data.tickets\[n\].name**

string

Nome do ingresso

**data.tickets\[n\].showOnEventPage**

boolean

Mostrar ingresso na página do evento

**data.tickets\[n\].isPaid**

boolean

Ingresso é pago

**data.tickets\[n\].isUnlimited**

boolean

Ingresso é ilimitado

**data.tickets\[n\].buyLimit**

number

Limite de ingressos por compra

**data.tickets\[n\].buyMinimum**

number

Compra mínima de ingressos por compra

**data.tickets\[n\].batches**

array of object

Dados dos lotes

**data.tickets\[n\].batches\[n\].name**

string

Nome do lote

**data.tickets\[n\].batches\[n\].name**

string

Nome do lote

**data.tickets\[n\].batches\[n\].quantity**

number

Quantidade de ingressos do lote

**data.tickets\[n\].batches\[n\].startSalesAt**

string

Data de início de vendas do lote

**data.tickets\[n\].batches\[n\].endSalesAt**

string

Data de final de vendas do lote

**data.tickets\[n\].batches\[n\].price**

object

Informações do valor e moeda do lote

**data.tickets\[n\].batches\[n\].price.currency**

string

Moeda do lote

**data.tickets\[n\].batches\[n\].price.value**

number

Preço do lote

**data.createdAt**

string

Data de criação do webhook
