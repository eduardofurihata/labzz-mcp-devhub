# Listar Vínculos

Obtém todos os ids de participantes vinculados a um marcador

**Required scopes:**blinket\_write

**Rate limit:**30 requests/minute

GET

https://api.eduzz.com/blinket/v1/attendance-tags/:id/attendance-list

### Path params

**id**

Id do marcador

### Response params (200)

**items**

array of string

Ids dos participantes vinculados ao marcador

### Status codes

Status

Descrição

200

Success

403

Este usuário não tem permissão

[![Swagger Logo](/_next/image?url=%2Flogos%2Fswagger.webp&w=256&q=75 "Swagger API Reference")](/swagger#/Blinket/getblinketv1attendancetagsidattendancelist)

### Exemplos

ShellJavaScript

```shell
1curl --request GET \
2	--url 'https://api.eduzz.com/blinket/v1/attendance-tags/cb655282-1903-402f-a02d-1f78cd388426/attendance-list' \
3	--header 'Accept: application/json' 
```

**Response**200 - Lista de participantes vinculados

```json
1{
2  "items": [
3    "9bf4dcae-2340-4433-9079-49b7d46396c3",
4    "9bf4e8c5-0dbb-4acb-8c4d-cf59a54b3ffc"
5  ]
6}
```
