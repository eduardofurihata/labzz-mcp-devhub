# Criando um WebHook

No menu 'Configurações' você encontrará a tela de configuração de webhooks.

![Listagem de webhooks](/docs/images/webhook/list-empty.png)

Clique em "+ Nova configuração" e preencha os seguintes campos:

*   **Nome da configuração:** Título da configuração.
*   **Chave de acesso:** Nesse campo, é possível escolher qual chave de autenticação será utilizada para assinar as mensagens que serão enviadas.
*   **URL para envio dos dados:** URL para onde os dados serão enviados. É importante realizar um teste para garantir que a URL esteja retornando um status HTTP válido (códigos 200 a 299).
*   **Máximo de eventos enviados em paralelo para a URL:** A Eduzz é pioneira no envio controlado de webhooks para os parceiros. Durante um grande lançamento, pode acontecer de a quantidade de envios tornar o seu sistema indisponível. Com essa configuração, você consegue definir um limite de envios e garante que sua aplicação não vai cair devido ao alto volume.
*   **Quais eventos você deseja receber?:** Você pode escolher quais eventos serão enviados para sua URL, de diversas aplicações.

![Preview Image](/docs/images/webhook/form-new-filled.png)

Clique em "Criar configuração".

![Preview Image](/docs/images/webhook/list-not-empty.png)

E sua configuração será salva com sucesso.
