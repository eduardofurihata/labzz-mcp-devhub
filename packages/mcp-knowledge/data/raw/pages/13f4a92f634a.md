# Ingresso impresso

Notifica quando o participante faz a impressão ou download de um ou mais ingressos

**Evento:**

blinket.attendances\_printed

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento (attendances\_printed)

**sentDate**

string

Data do envio do evento

**data**

object

Dados do evento, produtor, participante e ingresso

**data.producer**

object

Dados do produtor

**data.producer.id**

number

Id do produtor do evento

**data.producer.name**

string

Nome do produtor do evento

**data.producer.email**

string

Email do produtor do evento

**data.event**

object

Dados do evento

**data.event.id**

string

Id do evento

**data.event.name**

string

Nome do evento

**data.event.type**

string

Tipo do evento

**data.participants**

array of object

Dados dos participantes

**data.participants\[n\].id**

string

Id único deste ingresso

**data.participants\[n\].inviteKey**

number

Número do ingresso

**data.participants\[n\].name**

string

Nome do participante

**data.participants\[n\].email**

string

Email do participante

**data.participants\[n\].phone**

string

Telefone do participante

**data.participants\[n\].document**

object

Documento do participante

**data.participants\[n\].document.type**

string

Tipo de documento do participante

**data.participants\[n\].document.number**

number

Número do documento do participante

**data.participants\[n\].status**

string

Status do ingresso

**data.participants\[n\].manuallyCreated**

boolean

Ingresso foi criado manualmente

**data.participants\[n\].checkInAt**Nullable

string

Data que o checkin foi realizado

**data.participants\[n\].tags**

array of object

Marcadores do participante

**data.participants\[n\].tags\[n\].id**

string

Id da tag

**data.participants\[n\].tags\[n\].title**

string

Nome da tag

**data.participants\[n\].tags\[n\].color**

string

Cor da tag

**data.participants\[n\].customFields**

object

Campos customizados do participante

**data.participants\[n\].customFields.key**

string

Chave do campo customizado

**data.participants\[n\].customFields.value**

string

Valor do campo customizado

**data.createdAt**

string

Data de criação do webhook
