# Detalhes do Evento

Detalhes do evento

**Required scopes:**blinket\_read

**Rate limit:**30 requests/minute

GET

https://api.eduzz.com/blinket/v1/events/:id

Busca detalhes do evento na Blinket por id

### Path params

**id**

Id do evento

### Response params (200)

**id**

uuid

Id do evento

**userId**

uuid

Id do usuário dono do evento

**type**

enum

Tipo do evento

presential

Presencial

online

Online

live

Ao vivo

**title**

string

Nome do evento

**description**

text

Descrição do evento

**saleUrl**

url

URL do evento

**streamUrl**

url

URL do streaming

**reserveTime**

number

Tempo de reserva em minutos

**transmissionLinks**

array of string

Links de transmissão

**isLive**

boolean

Evento ao vivo

**files**

array of string

Arquivos em anexo

**activePrecheckout**

boolean

Ferramenta de pré-checkout ativa

**imageUrl**

url

URL da imagem do evento

**startAt**

datetime

Data de início do evento

**endAt**

datetime

Data do final do evento

**createdAt**

datetime

Data de criação do evento

**modifiedAt**

datetime

Data da última modificação do evento

**tickets**

array of object

Ingressos do evento

uuid

Id do ingresso

**tickets\[n\].eventId**

uuid

**tickets\[n\].name**

string

boolean

Ingresso é pago

boolean

Limite mínimo de compra de ingresso por usuário

**tickets\[n\].isUnlimited**

boolean

Ingresso está ativo

datetime

**tickets\[n\].modifiedAt**

datetime

**tickets\[n\].lots**

Lotes do ingresso

**tickets\[n\].lots\[n\].id**

uuid

**tickets\[n\].lots\[n\].ticketId**

uuid

string

boolean

datetime

datetime

datetime

**tickets\[n\].lots\[n\].modifiedAt**

datetime

**tickets\[n\].lots\[n\].eduzzProductId**

number

Id do lote na Eduzz
