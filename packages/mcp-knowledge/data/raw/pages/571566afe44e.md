# Listar minhas escolas

Listar minhas escolas

**Required scopes:**alpaclass\_team\_read

GET

https://api.eduzz.com/alpaclass/v1/producer/teams

### Query params

**sort**

string

Ordenação dos itens podendo ter várias ordenações, seguindo o seguinte padrão: campo:ordenação Ex.: name:asc,url:desc

### Response params (200)

**items**

array of object

Lista de escolas

**items\[n\].id**

string

Id da escola

**items\[n\].name**

string

Nome da escola

**items\[n\].url**

string

URL da escola

### Status codes

Status

Descrição

200

Success
