# Descrição

Screenshot da interface do Developer Hub da Eduzz mostrando a página de detalhes de um aplicativo OAuth.

A tela exibe:
- Menu lateral escuro com seções:
  - Início
  - WEBHOOK (Configurações, Histórico de envios, Estatísticas de webhooks, Chaves de acesso)
  - APLICATIVOS (Meus aplicativos, Estatísticas da API)
- Área principal com detalhes do aplicativo:
  - Breadcrumb de navegação
  - Título "Detalhes do aplicativo"
  - Botões "Gerar URL de Login" e "Editar"
  - Informações do app:
    - Client ID (UUID para OAuth)
    - Nome: "Aplicativo teste"
    - Status: "Produção" (badge verde)
    - Status da Moderação: "Pré Aprovado"
    - Descrição e Website configurados
    - Campos de Escopos (permissões)
  - Seção "Desenvolvedores" destacada em vermelho:
    - Tabela com email do desenvolvedor convidado
    - Botão "Adicionar" para convidar novos devs

Esta imagem documenta a tela de gerenciamento de aplicativos OAuth no Developer Hub da Eduzz.
