# Descrição

Screenshot de um modal de exibição de credenciais OAuth após criação de aplicativo no Developer Hub da Eduzz.

A tela exibe:
- Ícone de alerta amarelo (⚠️) indicando informação importante
- Título "Credenciais do seu Aplicativo"
- Mensagem de aviso: "Essas informações não serão exibidas novamente. Copie e coloque em um local seguro"
- Credenciais OAuth geradas:
  - **ClientID**: UUID único do aplicativo (f714ff76-8b2e-41c1-9dd2-0959522a5d55)
  - **Secret**: chave secreta longa para autenticação (64 caracteres hexadecimais)
- Botão "Copiar e Fechar" para copiar as credenciais e fechar o modal
- Design escuro (dark mode)

Esta imagem documenta o momento crítico após a criação de um aplicativo OAuth, onde as credenciais (Client ID e Secret) são exibidas apenas uma vez e devem ser copiadas imediatamente.
