# Comissão Processada

Notifica quando uma fatura é paga e as comissões são processadas

**Evento:**

myeduzz.commission\_processed

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento (commission\_processed)

**sentDate**

string

Data do envio do evento

**data**

object

Dados da fatura

**data.invoiceId**

string

ID da fatura

**data.createdAt**

string

Data de criação de comissionamento

**data.price**

object

Valor da comissão

**data.price.currency**

number

Moeda do valor total da transação

**data.price.value**

number

Valor total da transação

**data.commissions**

object

Dados das comissões

**data.commissions.producer**

object

Dados da comissão do produtor

**data.commissions.producer.name**

string

Nome do produtor

**data.commissions.producer.id**

string

ID do produtor

**data.commissions.producer.commission**

object

Valor da comissão do produtor

**data.commissions.producer.commission.currency**

number

Moeda do valor da comissão do coprodutor

**data.commissions.producer.commission.value**

number

Valor da comissão do coprodutor

**data.commissions.coproducers**

array of object

Dados da comissão do(s) coprodutor(es)

**data.commissions.coproducers\[n\].name**

string

Nome do coprodutor

**data.commissions.coproducers\[n\].id**

string

ID do coprodutor

**data.commissions.coproducers\[n\].commission**

object

Valor da comissão do coprodutor

**data.commissions.coproducers\[n\].commission.currency**

number

Moeda do valor da comissão do coprodutor

**data.commissions.coproducers\[n\].commission.value**

number

Valor da comissão do coprodutor

**data.commissions.affiliates**

array of object

Dados da comissão do(s) afiliado(s)

**data.commissions.affiliates\[n\].name**

string

Nome do afiliado

**data.commissions.affiliates\[n\].id**

string

ID do afiliado

**data.commissions.affiliates\[n\].commission**

object

Valor da comissão do afiliado

**data.commissions.affiliates\[n\].commission.currency**

number

Moeda do valor da comissão do afiliado

**data.commissions.affiliates\[n\].commission.value**

number

Valor da comissão do afiliado
