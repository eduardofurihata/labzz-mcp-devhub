# Último acesso do aluno no curso

Notifica o último acesso do aluno no curso

**Evento:**

alpaclass.student\_course\_last\_seen

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento (student\_course\_last\_seen)

**sentDate**

string

Data do envio do evento

**data**

object

Dados do evento

**data.student**

object

Dados do aluno

**data.student.id**

string

Id do aluno na escola

**data.student.name**

string

Nome do aluno

**data.student.email**

string

E-mail do aluno

**data.course**

object

Dados do curso

**data.course.id**

string

Id da escola

**data.course.name**

string

Nome da escola

**data.team**

object

Dados da escola

**data.team.id**

string

Id da escola

**data.team.name**

string

Nome da escola
