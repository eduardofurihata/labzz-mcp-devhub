# Criar Aluno

Cria um novo aluno em uma escola.

**Required scopes:**alpaclass\_team\_write

POST

https://api.eduzz.com/alpaclass/v1/producer/teams/:teamId/students

### Path params

**teamId**

Id da escola

### Body params

**name**

string

Nome do aluno

**email**

string

Email do aluno

**phone**

string

Telefone do aluno

**document**

string

CPF do aluno

**tags**

array of string

Tags do aluno

### Response params (201)

**id**

string

Id do aluno

**name**

string

Nome do aluno

**email**

string

Email do aluno

**phone**

string

Telefone do aluno

**document**

string

Documento do aluno

### Status codes

Status

Descrição

201

Success
