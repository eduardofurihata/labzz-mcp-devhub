# Contrato Criado

Notifica quando um contrato é criado.

**Evento:**

myeduzz.contract\_created

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento

**data**

object

Dados do evento

**data.producer**

object

Informações do produtor.

**data.producer.id**

string

Id do produtor.

**data.producer.name**

string

Nome do produtor.

**data.producer.email**

string

Email do produtor.

**data.products**

array of object

Informações dos produtos.

**data.products\[n\].id**

string

Id do produto.

**data.products\[n\].name**

string

Nome do produto.

**data.products\[n\].deliveries**

array of string

Formas de entrega do produto.

**data.products\[n\].plan**

object

Plano do produto.

**data.products\[n\].plan.id**

string

Id do plano.

**data.products\[n\].plan.name**

string

Nome do plano.

**data.products\[n\].price**

object

Preço do produto.

**data.products\[n\].price.currency**

string

Moeda usada no produto.

**data.products\[n\].price.value**

number

Valor do produto.

**data.products\[n\].membershipFee**

object

Taxa de adesão.

**data.products\[n\].membershipFee.currency**

string

Moeda da taxa de adesão do produto.

**data.products\[n\].membershipFee.value**

number

Valor da taxa de adesão do produto.

**data.contract**

object

Informações do contrato.

**data.contract.id**

string

Id da fatura.

**data.contract.payment**

object

Informações de pagamento.

**data.contract.payment.method**

string

Forma de pagamento.

**data.contract.payment.totalOfInstallments**

number

Número de parcelas.

**data.contract.status**

enum

Status do contrato.

upToDate

Em dia

awaitingPayment

Aguardando pagamento

late

Atrasado

canceled

Cancelado

defaulter

Inadimplente

suspended

Suspenso

trial

Trial

finished

Finalizado

free

Gratuito

**data.contract.trialDays**

number

Dias de teste.

**data.contract.createdAt**

string

Data de criação do contrato.

**data.contract.updatedAt**

string

Data de atualização do contrato.

**data.contract.contentAccess**

object

Informações de acesso de conteúdo.

**data.contract.contentAccess.removeOnContractEnd**

boolean

Remover acesso ao conteúdo no final do contrato.

**data.contract.contentAccess.removeOnLatePayment**

boolean

Remover acesso ao conteúdo de pagamento com atraso.

**data.contract.isUnlimitedInstallments**

boolean

Se o contrato é PSL (parcelamento sem limite).

**data.contract.recurrence**

object

Informações da recorrência.

**data.contract.recurrence.startsAt**

string

Data de inicio do contrato.

**data.contract.recurrence.nextDue**

string

Data do próximo vencimento.

**data.contract.recurrence.currentDue**

string

Data atual do contrato.

**data.contract.recurrence.finalDue**

string

Data do último vencimento.

**data.contract.recurrence.finishesAt**

string

Data do término do contrato.

**data.contract.recurrence.frequency**

object

Informações da frequência.

**data.contract.recurrence.frequency.type**

string

Tipo de frequência da cobrança.

**data.contract.recurrence.frequency.value**

number

Valor referente a frequência da cobrança.

**data.contract.recurrence.charges**

object

Informações da cobrança.

**data.contract.recurrence.charges.total**

number

Número de cobranças.

**data.contract.recurrence.charges.current**

number

Moeda usada nas cobranças.

**data.contract.recurrence.isFinite**

boolean

Se é finito ou infinito.

**data.contract.recurrence.price**

object

Informações de preço.

**data.contract.recurrence.price.currency**

string

Moeda usada no preço.

**data.contract.recurrence.price.value**

number

Valor do contrato.

**data.customer**

object

Informações do cliente.

**data.customer.name**

string

Nome do cliente.

**data.customer.email**

string

Email do cliente.

**data.customer.phone**

object

Informações do telefone do cliente.

**data.customer.phone.countryCode**

string

Código de telefone do país do cliente.

**data.customer.phone.areaCode**

string

Codigo de área do cliente.

**data.customer.phone.number**

string

Número de telefone do cliente.

**data.financialResponsible**

object

Informações do responsável financeiro.

**data.financialResponsible.name**

string

Nome do responsável financeiro

**data.financialResponsible.email**

string

Email do responsável financeiro

**data.financialResponsible.phone**

object

Informações do telefone do responsável financeiro.

**data.financialResponsible.phone.countryCode**

string

Código de telefone do país do responsável financeiro.

**data.financialResponsible.phone.areaCode**

string

Código de área do responsável financeiro.

**data.financialResponsible.phone.number**

string

Número de telefone do responsável financeiro.

**data.sentDate**

string

Data do envio do evento
