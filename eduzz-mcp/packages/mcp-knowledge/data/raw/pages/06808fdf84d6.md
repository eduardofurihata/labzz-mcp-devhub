# Ingresso Cancelado

Notifica quando um ingresso é cancelado

**Evento:**

blinket.attendance\_canceled

### Response params

**id**

string

Id do evento

**event**

string

Nome do evento (attendance\_canceled)

**sentDate**

string

Data do envio do evento

**data**

object

Dados do evento, produtor, participante e ingresso

**data.producer**

object

Dados do produtor

**data.producer.id**

number

Id do produtor do evento

**data.producer.name**

string

Nome do produtor do evento

**data.producer.email**

string

Email do produtor do evento

**data.event**

object

Dados do evento

**data.event.id**

string

Id do evento

**data.event.name**

string

Nome do evento

**data.event.type**

string

Tipo do evento

**data.participant**

object

Dados do participante

**data.participant.id**

string

Id único deste ingresso

**data.participant.inviteKey**

number

Número do ingresso

**data.participant.name**

string

Nome do participante

**data.participant.email**

string

Email do participante

**data.participant.phone**

string

Telefone do participante

**data.participant.status**

string

Status do ingresso

**data.participant.manuallyCreated**

boolean

Ingresso foi criado manualmente

**data.participant.tags**

array of object

Marcadores do participante

**data.participant.tags\[n\].id**

string

Id da tag

**data.participant.tags\[n\].title**

string

Nome da tag

**data.participant.tags\[n\].color**

string

Cor da tag

**data.participant.customFields**

object

Campos customizados do participante

**data.participant.customFields.key**

string

Chave do campo customizado

**data.participant.customFields.value**

string

Valor do campo customizado

**data.createdAt**

string

Data de criação do webhook
