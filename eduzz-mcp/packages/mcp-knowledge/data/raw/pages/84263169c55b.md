# Listar entregas da escola

Listar as entregas da escola

**Required scopes:**alpaclass\_delivery\_read

GET

https://api.eduzz.com/alpaclass/v1/producer/teams/:teamId/deliveries

### Path params

**teamId**

Id da escola

### Query params

**page**

number

Página atual

**itemsPerPage**

number

Número máximo de itens por página

**sort**

string

Ordenação dos itens podendo ter várias ordenações, seguindo o seguinte padrão: campo:ordenação Ex.: name:asc,platform:desc

### Response params (200)

**pages**

number

Número de páginas

**page**

number

Página atual

**itemsPerPage**

number

Número máximo de itens por página

**totalItems**

number

Número total de items

**items**

array of object

Entragas da escola

**items\[n\].id**

string

Id da entrega

**items\[n\].name**

string

Nome da entrega

**items\[n\].platform**

enum

Nome da plataforma

eduzz

Eduzz

dmg

Digital Manager Guru

hotmart

Hotmart

kiwifi

Kiwify

tmb

TMB

api

API

**items\[n\].isActive**

boolean

Identifica se a entrega esta ativa

**items\[n\].isSubscription**

boolean

Identifica se a entrega é assinatura

### Status codes

Status

Descrição

200

Success
