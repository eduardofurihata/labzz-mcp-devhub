# Alterar Marcador

Altera um marcador existente

**Required scopes:**blinket\_write

**Rate limit:**30 requests/minute

PUT

https://api.eduzz.com/blinket/v1/attendance-tags/:id

### Path params

**id**

Id do marcador

### Body params

**title**

string

Novo título do marcador

**color**

string

Nova cor do marcador em hexadecimal

### Response params (200)

**id**

string

Id do marcador alterado

**title**

string

Título

**color**

string

Cor

**createdAt**

datetime

Data de criação do marcador

**modifiedAt**

datetime

Data da última modificação do marcador

### Status codes

Status

Descrição

200

Success

403

Este usuário não tem permissão

422

Erro de validação

[![Swagger Logo](/_next/image?url=%2Flogos%2Fswagger.webp&w=256&q=75 "Swagger API Reference")](/swagger#/Blinket/putblinketv1attendancetagsid)

### Exemplos

ShellJavaScript

```shell
1curl --request PUT \
2	--url 'https://api.eduzz.com/blinket/v1/attendance-tags/:id' \
3	--header 'Accept: application/json' \
4	--data '
5{
6  "title": "New Title",
7  "color": "#00FF00"
8}
9'
```

**Response**200 - Marcador alterado com sucesso

```json

```
