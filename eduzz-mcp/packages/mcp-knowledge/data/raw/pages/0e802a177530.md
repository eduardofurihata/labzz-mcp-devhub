# Criar Marcador

Cria um novo marcador que poderá ser usado em diferentes eventos

**Required scopes:**blinket\_write

**Rate limit:**30 requests/minute

POST

https://api.eduzz.com/blinket/v1/attendance-tags

### Body params

**title**

string

Título do novo marcador

**color**

string

Cor do novo marcador em hexadecimal

### Response params (201)

**id**

string

Id do novo marcador criado

**title**

string

Título

**color**

string

Cor

**createdAt**

datetime

Data de criação do marcador

**modifiedAt**

datetime

Data da última modificação do marcador

### Status codes

Status

Descrição

201

Success

403

Este usuário não tem permissão

422

Erro de validação

[![Swagger Logo](/_next/image?url=%2Flogos%2Fswagger.webp&w=256&q=75 "Swagger API Reference")](/swagger#/Blinket/postblinketv1attendancetags)

### Exemplos

ShellJavaScript

```shell
1curl --request POST \
2	--url 'https://api.eduzz.com/blinket/v1/attendance-tags' \
3	--header 'Accept: application/json' \
4	--data '
5{
6  "title": "XYZ",
7  "color": "#FF0000"
8}
9'
```
